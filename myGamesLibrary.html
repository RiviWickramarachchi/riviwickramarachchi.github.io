MyGames Library

About the Project

This website was developed by myself as a passion project for me to gain hands-on experience working on web applications using the ASP.NET framework in C#. The game data used in the application were retrieved through the IGDB database, which is a database for video games that is freely available to the public.

Features

  • A User Login system
  • A search feature to look for game titles
  • A feature to maintain a library of games that the user has played
