Fast Reader

About the Project

This web application was developed as a passion project to learn and gain hands on experience working with the Python Django Framework. Fast Reader works as an application that helps people to read documents easily. When the user uploads a word or a pdf document, the application begins to display the contents of the document word by word making it easier for the user to read it. Furthermore, if the the speed of which the words are displayed is too fast, the user can adjust the speed according to his/her preference.

Features

  • A file uploading system
  • A method to change the word displaying speed
  • Functionalities to read the uploaded pdf or word files
