Mr Froggo

About the game

Mr Froggo is a hypercasual mobile game that was developed by me and a friend of mine as a passion project. In this game, you play as a frog who tries to survive by catching various insects that swarm around you. The game has a number of insects that give different types of buffs and nerfs to the player and the difficulty of the game is slowly increased with the introduction of new types of insects. Mr froggo is currently in the beta testing stage and more features are to be introduced in the future...

Responsibilities

  • Programming scene transitions
  • Adding UI components to the game
  • Designing game mechanics, buffs and nerfs in game
  • Programming the core game mechanics and gameplay elements
  • Adding background, button click and bug capture event sounds to the game
  • Implementing an in game store to purchase different backgrounds using in game currency
