Cat's Eye

About the game


A cute first person adventure horror game themed on cats that was created by a group of extremely talented individuals as a submission for the Global Game Jam 2022 which was held in January. In this game the protagonist find out that his furry friend is missing and he needs to collect nine cat lives scattered across the forest in order to bring his pet back. The player is given a crystal stone which he can use to enter the spirit world in order to collect the nine lives. But in the spirit world the player may encounter evil cat spirits guarding the cat lives who can kill him in an instance.

Responsibilities

  • Creating UI for the Win and Game Over screens
  • Programming the scene transitions
  • Programming the Environment transitions

Instructions To Install

  • Visit The Global Game Jam website and download the 'CatsEye.zip' file.
  • Unzip the file and run the 'Cats Eye.exe' executable.
  • Enjoy the game.
